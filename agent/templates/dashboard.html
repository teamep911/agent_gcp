    Chưa có phân tích nào

    Tính năng Claude AI Analysis đang trong quá trình phát triển. Sau khi hoàn thiện, lịch sử và kết quả phân tích sẽ hiển thị tại đây.

    Kiến trúc tích hợp
    Data Flow
    Luồng dữ liệu từ Oracle → Claude → Dashboard
    1. TriggerOEM Alert Critical → Agent captures Perf Bundle
    2. Analysis RequestAgent gửi AWR/ASH data → Claude API (claude-sonnet-4-6)
    3. Insight GenerationClaude phân tích → RCA + Recommendations JSON
    4. DeliveryInsights → Google Chat + Dashboard (tab này)

    Cấu hình Claude AI dự kiến
